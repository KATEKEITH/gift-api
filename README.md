## 요구 사항

1. 선물하기 결제 후 구매자와 수령자에게 알림이 발송된다
2. 수령자는 선물을 수락 or 거절 할 수 있다
3. 수령자가 수락한 경우 본인의 배송지 주소를 입력하면 배송이 시작된다
